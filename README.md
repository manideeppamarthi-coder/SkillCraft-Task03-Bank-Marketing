# SkillCraft Data Science Internship – Task 03

## Bank Marketing Prediction using Decision Trees

### Objective

Build a Decision Tree Classifier to predict whether a customer will subscribe to a term deposit based on demographic and campaign-related information.

### Dataset

Custom Bank Marketing Dataset (`bank_marketing_custom_dataset.xlsx`)

### Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

### Project Workflow

1. Loaded and explored the dataset.
2. Preprocessed categorical features.
3. Split data into training and testing sets.
4. Built a Decision Tree Classification model.
5. Evaluated model performance using accuracy and classification metrics.
6. Visualized the decision tree.

### Machine Learning Concepts

* Data Preprocessing
* Label Encoding
* Train-Test Split
* Decision Tree Classification
* Model Evaluation
